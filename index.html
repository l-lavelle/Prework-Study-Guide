HTML

  • Used to create the structure of a webpage
  • Made up of elements that work together to provide content and basic framework for all webpages
    • Elements perform unique roles to divide the content on the page and describe it to the browsers and other developers
    • Elements are composed of opening and closing tags
    • Attributes define additional information about elements
  • The head element contains information about the webpage.
    • Elements included in the head are not visible when a user looks at a webpage
    • Provides important information to browser, search engine, and developers
  • The body element represents the visible content shown to the user.
    • Broken down into header, main, footer elements creating different sections
    • Want to use Semantic HTML elements when possible
